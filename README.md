## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/myra-labs/myra.git
   cd myra
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
